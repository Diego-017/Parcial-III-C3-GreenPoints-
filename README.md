Parcial-3-C3-GreenPoints

GreenPoints es una aplicación web desarrollada con Django que permite registrar dispositivos de reciclaje, gestionar entradas de residuos y asignar puntos como incentivo ambiental. El sistema integra autenticación de usuarios, generación automática de códigos únicos, envío de notificaciones por correo y un panel interactivo que muestra métricas e indicadores ambientales basados en datos reales. Su objetivo es fomentar la participación ciudadana en el reciclaje y ofrecer una plataforma moderna, intuitiva y sostenible para el manejo de información ambiental.
